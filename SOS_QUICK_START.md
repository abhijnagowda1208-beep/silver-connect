# SOS Feature - Quick Start (30 seconds)

## ⚡ Quick Setup

### 1️⃣ Add Emergency Contact
- Profile → Safety Network → Manage → "Add New Contact"
- Enter name, relationship, phone number

### 2️⃣ Set as Primary
- Click "Set as Primary" on your contact
- You'll see "SOS RECIPIENT" badge in red

### 3️⃣ Test SOS
- Go to Dashboard
- Click the red **SOS** button
- Confirm the alert

## ✅ Now You're Ready!

Your emergency contact will receive alerts when you press SOS.

---

## 📊 View Your SOS History
- Profile → Safety Network → "History"
- See all alerts you've triggered

## 🆘 Remember
**Always set a PRIMARY contact before using SOS!**
Without a primary contact, SOS won't work.
